# Teacher guide

The **Teacher** portal covers academics plus a fleet-availability view for live tracking.

Portal: `/teacher` · Sign in as `teacher1` / `Password@123`.

## Your sections

| Section | Purpose |
| --- | --- |
| Overview | Your teaching dashboard |
| Attendance | Mark/view class attendance |
| Homework / Study material | Assign homework and upload resources |
| Evaluate | Enter exam marks per student (`/teacher/evaluate`) |
| Tracking | Live fleet snapshot + per-driver route detail |
| Notices | School-wide notice board |
| Profile | Your teaching account |
| Academics (via menu) | Timetable, homework, study materials, exams/results as available |

## Example — Check bus availability and a route

1. Go to **Tracking**.
2. See fleet stats: online drivers, running trips, active buses.
3. Select a driver to view their assigned **route with all stops** on the map and in the route panel,
   including live bus position and ETA when a trip is running.

## Example — Mark class attendance

1. Go to **Attendance**.
2. Pick the class/date, mark each student `PRESENT`, `ABSENT`, `LATE`, or `EXCUSED`, and save.
   One record per student per day is kept.

## Example — Enter exam results

1. Go to **Evaluate**.
2. Select an exam. Students in that exam's grade/section load into a grid.
3. Enter each student's **marks** (out of the exam's max), an optional **grade letter** and **remarks**, then
   tap **Save** on the row. Results publish to the student/parent **Results** page.

## Notes

- Everything is scoped to your school automatically.
- Notices target audiences such as ALL, STUDENTS, TEACHERS, PARENTS, or STAFF.
